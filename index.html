Selectores de id

Un id es un identificador para etiquetas único e irrepetible, algo así como el DNI que tiene cada persona. Dentro de CSS se puede identificar un elemento con el atributo id y aplicarle un estilo a un único, para esto se usa el símbolo # seguido del valor de id del elemento.

Seudo clases

Una seudo-clase o pseudo-clase es un selector que se encuentra en un estado particular. Por ejemplo, cuando el cursor del mouse toque el e lemento HTML se le aplicará un estilo específico.
